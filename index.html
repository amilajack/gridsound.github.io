DAW

The GridSound 's DAW is a work-in-progress free open-source HTML5 Digital Audio Workstation based on the new fresh Web Audio technology. This project is compatible with any up to date modern browsers like : Chrome, Firefox, Safari.

All the project is open source, this means you have free access to read, review and contribute to the whole code at GitHub here.

Use it for free without any limit !

Web Audio - library

With this standalone JavaScript library we extract every reusable Web Audio code. We offer encapsulation for buffers, samples, array of effects, composition.

See it on GitHub !
